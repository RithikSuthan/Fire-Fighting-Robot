# Fire-Fighting-Robot

Software Requirements:
•	Arduino IDE
•	Blynk

Hardware Requirements:
•	Arduino UNO, nodeMCU-esp8266
•	IR Flame sensor 
•	L298N motor driver module
•	Submersible water pump
•	Single Channel Relay Module(5v)
•	BO Motors (3v -12v)
•	Chassis
•	Adaptor(12v)
•	Switch
•	Jump wires
•	Servo motor
